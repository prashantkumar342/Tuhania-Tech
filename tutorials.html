What is Linux ?

Linux is an operating system just like iOS, Android, Windows. Linux is free and open source. In fact, one of the most popular Operating system on the planet, the Android, is powered by the Linux OS. Operating System is an type of Software that manages all the system resources associated with our system.


The main parts or services with Linux developed and run :--


(1.) Bootloader - This is also an system software that manages all boot processes of you linux system. boot up your system. Bootloader starts your operating system and starts requried Daemons or services.

(2.) Kernel - This is the one piece of whole that is actually called 'Linux'. The kernel is the core of the Operating System and manages the CPU, memory, and peripheral devices. The kernel is the lowest leve of the Operating System

(3.) Init System - This is a sub-system that bootstraps the user space and is charged with controlling daemons. One of the most widely used init systems is systemd, which also happens to be one of the most controversial. It is the init system that manages the boot process, once the initial booting is handed over from the bootloader.

(4.) Daemons - These are background serives that run in Operating System Such as printing, sound, fax, search indexing or other device drivers and other internet related background services.

(5.) Graphical Server - This is the sub system that renders something on display monitor. It is commonly referred to as the X server or just X.

(6.) Desktop Environment - This is the main piece of graphical user intreaction with linux. using this user can operate the linux operating system because linux do not have its own user interface in most of the case linux distributions are made with a desktop manage that provide us a graphical user interface with the help of this we can able to use or operate the linux ditribution by Graphical User interface. some most popular desktop Environment is:

  • Gnome
  • KDE Plasama
  • Mint etc.

(7.) Applications - Desktop environments do not offer the full array of software applications. Just like macOS, Windows, Linux offers thousands upon thousands high-quality softwares titles that can easily found and installed by packge manager or app store. Most modern Linux distributions includes App Store-like tools that centralize and simply application installation. For example, ubunt Linux has the Ubuntu Software Center (a rebran of GNOME Software) which allows you to quickly search among the thousand of apps and install them from one centralizes location.

What is an Operating System ?

An Operating System is a piece of software that manages all the resources of a computer system both hardware and software and provides an environment in which users can perform tasks by executing the applications or processes in a convenient and efficient manner. OS hides the complexity of the hardware by working as resource manager.

Layered Structure Of operating System :

  • Layer 1 - Computer Hardware
  • Layer 2 - CPU Scheduling
  • Layer 3 - Memory Management
  • Layer 4 - Process Management
  • Layer 5 - Operating System
  • Layer 6 - User Program or Application Softwares

Objectives of OS

The main targets of an operating system are as follows :

  • Convenience - An operating system improves the use of a Hardware. Operating systems gives the ability to users to use systems without the stress of configuring system and resource management.
    • br
  • Efficiency - An Operating System has the ability to manage all the resources properly and use them as needed. This is due to less time spent on configuring the system.

  • Ability to evolve - An operating system should be designed in such a way that it allows for the effective development, testing, and introduction of new features without interfering with service.

  • Management Of System Resources - Operating systems are able to share resources fairly among various processes that are running and user.

Functions Of Operating System

  1. Memory management :

    Operating System Manages the primary or main memory. If any program is executed, it has to be present in the memory. Main memory is faster than Secondary memory and directly accessed by the CPU. When a program is executed the operating system allocates the required memory to the process of it and by the CPU it handles the process in an efficient manner after the completion of the process Operating System deallocates the memory and free up the space in main memory for another process.


    Operating System :
    • Allocates and Deallocates the memory
    • Keeping the record of which part of memory is used and which process is using.
    • Distributing the memory while multiprocessing etc.

  2. CPU Management or Scheduling :

    Every software that runs on a computer, whether in the background or in the frontend, is a process. Processor management is an execution unit in which a program operates. The operating system determines the status of the processor and processes, selects a job and its processor, allocates the processor to the process, and de-allocates the processor after the process is completed

    When more than one process runs on the system the OS decides how and when a process will use the CPU. Hence, the name is also CPU Scheduling.
    The OS :

    • Manages the CPU uses by process.
    • Keeps the record of CPU status.

    Certain algorithms used for CPU Scheduling in OS are as follows :
    • First Come First Out.
    • Shortest Job First.
    • Round-Robin Scheduling.
    • Priority-based scheduling etc.

  3. File Management :
    The operating system manages resource allocation and deallocation. It specifies which process receives the file and for how long. It also keeps track of information, location,uses,status, and so on. These groupings of resources are referred to as file systems. The files on a system are stored in different directories.
    The OS:
    • Keeps records of the status of and locations of files.
    • Allocates and deallocates.
    • Decides who gets the resources

  4. Storage management :
    Storage Management is defined as it refers to the management of the data storage equipment’s that are used to store the user/computer generated data. Hence it is a tool or set of processes used by an administrator to keep your data and storage equipment’s safe. Storage management is a process for users to optimize the use of storage devices and to protect the integrity of data for any media on which it resides and the category of storage management generally contain the different type of subcategories covering aspects such as security, virtualization and more, as well as different types of provisioning or automation, which is generally made up the entire storage management software market.

    The OS uses storage management for :
    • Improving the performance of the data storage resources.
    • It Optimizes the use of various storage devices.
    • Assists businesses in storing more data on exiting hardware, speeding up the data retrieval process, preventing, preventing data loss, meeting data retention regulations, and lowering IT costs.
